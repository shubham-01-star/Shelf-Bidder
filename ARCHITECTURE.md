## рҹ“Ҡ Technology Stack

### Frontend
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **PWA:** Service Worker, Offline Support
- **State:** React Hooks, Context API

### Backend
- **Runtime:** Node.js 18+
- **Framework:** Next.js API Routes
- **Language:** TypeScript
- **Validation:** Zod

### Database
- **Primary:** PostgreSQL 15
- **Deployment:** Docker Container
- **Features:** ACID Transactions, Row-level Locking
- **Connection:** pg (node-postgres)

### AWS Services
- **Bedrock:** AI Vision Analysis (Nova Pro, Nova Lite, Claude Haiku)
- **S3:** Photo Storage (Mumbai region)
- **Cognito:** User Authentication
- **Region:** ap-south-1 (Mumbai)

### External Services
- **Resend:** Email Service (OTP, Welcome emails)
